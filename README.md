# Solid Mind GPT
This application is use the new GPT-3.5-turbo model from openai to generate mindmaps from your text input. The mindmap is generated by using the [MarkMap](https://markmap.js.org/) library.

## Tech Stack
- SolidJS
- SolidStart
- TailwindCSS (DaisyUI)
- Vercel
- Vite

## Possibilities
- Guided mindmap
- Free mindmap
- Decision tree
- Summary
